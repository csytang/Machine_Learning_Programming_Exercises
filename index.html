W2_Linear Regression

In this exercise, linear regression along with gradient descent is implemented and applied to real life dataset. Specifically, the linear regression model is employed to predict the profit of a restaurant franchinse in a new city given the proft-population data. Gradient Descent is applied to optimal performance and minimize the cost function For exercise "Linear regression with multiple variable", by learning from historical housing price data in Portland, Oregon, the learning hyphothesis is able to predict housing price with given number of bedrooms, and size of the house

W3_Logistic Regression

In this exercise, logistic regression is implemented and applied to two real life dataset. Concretely, the logistic regression model is applied to predict the probability of admission given two historial exam data. Furthermore, regularized logistic regression is used to solve overfitting problem when designing hypothesis for the problem. A deeper touch on regularized logistic regression is that using different regularize parameter(lambda) to see the influence of that parameter

W7_Support Vector Machine

In this exercise,support vector machine is applied to build email spam classifer, at the first step, we train our SVM using Gaussian Kernal, and then we load the training examples and still use that SVM to classify the emails. When processing email, the preprocessing procedure is cricial since, it will regularized all training examples regardless of specific cases. Specifically, the normalization steps include: lower-casing, strippint HTML, normalizing URLs, normalizing email addresses, normalizing numbers, normalizing dollars,word stemming, removal of non-words. Featured vocabulary list is built and training examples(emails) are mapped to index vector with y to indicate it is spam or not. In addition, try the opens source spam email database and test the performance.
